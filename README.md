# สมุดกำไรแม่ค้า — Shopee Profit Calculator

## รันบนเครื่องตัวเอง
1. ติดตั้ง Node.js (https://nodejs.org) เวอร์ชัน 18 ขึ้นไป
2. เปิด terminal ในโฟลเดอร์นี้แล้วรัน:
   npm install
   npm run dev
3. เปิด http://localhost:5173

## Deploy ขึ้นเว็บฟรี (Vercel)
1. สมัคร https://vercel.com (ใช้ GitHub login)
2. อัปโหลดโฟลเดอร์นี้ขึ้น GitHub repo
3. ใน Vercel กด Add New > Project > เลือก repo นี้ > Deploy
4. ได้ URL เช่น your-app.vercel.app ใช้งานได้เลย
